# 📊 Dashboard de Análisis de Ventas

Este proyecto consiste en el desarrollo de un dashboard interactivo enfocado en el análisis de ventas, utilizando Power BI como herramienta principal de visualización.

## 🎯 Objetivo

Analizar el comportamiento de las ventas a lo largo del tiempo, identificar tendencias y generar indicadores clave (KPIs) que apoyen la toma de decisiones estratégicas.

## 📌 Indicadores Clave (KPIs)

- 💰 Ventas Totales: $109.81 millones  
- 📉 Costos Totales: $97.26 millones  
- 📈 Ganancias: $12.55 millones  
- 📦 Unidades Vendidas: 275 mil  

## 📊 Visualizaciones incluidas

- 📈 Análisis de ventas y costos por año
- 📈Análisis de ventas por trimestre
- 📈Análisis de ventas por canales de pago
- 🥧 Distribución porcentual de ganancias anuales  
- 📊 Comparación de desempeño financiero  
- 🎯 Segmentación por diferentes filtros (fecha, país, categoría, producto, etc.)

## 🛠️ Tecnologías utilizadas

- Pentaho (Extracción de data)
- Power BI  
- PostgresSQL (fuente de datos)  
- SQL (para consultas y manejo de datos, si aplica)
